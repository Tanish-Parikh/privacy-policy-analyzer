Privacy Policy Analyzer

Privacy Policy Analyzer is a browser extension that scans website privacy policies and highlights potentially risky clauses related to data sharing, tracking, and third-party access.

The extension helps users quickly understand how their data may be used by simplifying complex legal language and identifying important privacy risks.

Installation / Download Guide

Follow these steps to install the Privacy Policy Analyzer extension manually.

Step 1: Download the Repository

Clone the repository using Git:

git clone https://github.com/Tanish-Parikh/privacy-policy-analyzer.git

Or download it manually:

Go to the repository page

Click the Code button

Click Download ZIP

Extract the ZIP file

Step 2: Open Chrome Extensions Page

Open Google Chrome and go to:

chrome://extensions/
Step 3: Enable Developer Mode

Turn Developer Mode ON (top-right corner).

Step 4: Load the Extension

Click Load unpacked

Select the extracted project folder

Example folder:

privacy-policy-analyzer
Step 5: Extension Installed

The Privacy Policy Analyzer extension will now appear in your browser toolbar and is ready to use.
